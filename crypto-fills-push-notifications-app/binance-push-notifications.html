How to receive push notifications for trades and order fills on Binance Spot and Futures

1. Download the Crypto Fills iOS app and complete the "Get Started" step, giving permission for the app to send push notifications

2. Tap the '+' button in the top right, then select the Binance Spot or Futures option, depending on which exchange you are trading on

3. In Binance, generate API keys in your account settings (if possible, ensure these are read-only API keys)

4. Copy and paste your API key and secret into the Crypto Fills app

5. Tap "Test", and when it succeeds, tap "Save"
